# 🎲 Dice Rolling Simulator

## 📖 Description
This is a simple Dice Rolling Simulator built using Python. It generates a random number between 1 and 6, just like rolling a real dice.

## ✨ Features
- Roll a virtual dice
- Generates a random number from 1 to 6
- Easy to use
- Beginner-friendly project

## 💻 Technologies Used
- Python

## 🚀 How to Run
1. Run the `dice_rolling_simulator.py` file.
2. Press Enter to roll the dice.
3. View the random dice number.
